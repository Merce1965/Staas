This is a Beta prototype for the Defence Service Manual project to help projects meet MOD and government standards. Please be aware that the prototype's main purpose is for use in user research sessions to understand what does and doesn't work for users.

! Please note All pages and features within this prototype should be considered EXPERIMENTAL.

Features included with this prototype:

  • A Service Manual style front page where users can access everything in one place
  • Guidance grouped into themes to help users navigate to relevant content
  • Step by step guidance to identify specific activities required to satisfy each area (This feature needs further investigation)
