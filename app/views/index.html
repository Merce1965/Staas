This is a Beta prototype for the Standards as a service project to help projects meet MOD and government standards. Please be aware that the prototype's main purpose is for use in user research sessions to understand what does and doesn't work for users.

! Please note All pages and features within this prototype should be considered EXPERIMENTAL.

Features being tested with this prototype include:

  • A Service Manual style front page where users can access everything in one place
  • Standards grouped into themes to help users navigate the numerous rules
  • Step by step guidance to identify specific activities required to satisfy each area
